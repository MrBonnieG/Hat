# 🎩 Hat

**Hat** is a simple and highly configurable `/hat` plugin that allows players to wear any item from their hand as a hat with permissions support and customizable messages.

---

## ✨ Features

- 🎩 Wear any item as a hat with `/hat`
- 🔐 Permission-based system
- 📦 Material whitelist support
- ⚡ Lightweight and fast

---

## 📦 Commands
| Command | Description |
|:----------------|:------------------------------|
| `/hat` | Puts the item in your hand on your head |
| `/hat reload` | Reloads the plugin configuration |
## 📋 Permissions
| Permission | Description |
|:----------------|:---------------------|
| `hat.use` | Allows using the `/hat` command |
| `hat.reload` | Allows reloading the configuration |
